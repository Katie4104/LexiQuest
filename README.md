LexiQuest – Language Learning Adventure Game 

Description 

LexiQuest is an interactive language learning game designed to help users develop practical language skills through immersive gameplay. The application places players in real-world scenarios such as airports, train stations, and marketplaces, where they must complete language-based challenges to progress. 

The game emphasizes context-driven learning, allowing users to acquire vocabulary and comprehension naturally rather than through memorization. Players can either create an account to track their progress or continue as a guest. Through exploration and decision-making, users engage with multiple languages in a structured yet engaging environment. 


Key features include: 


  User authentication system (login and registration)  

  Scene-based navigation (splash screen, login, main menu, levels)  

  Interactive map with scenario-based progression  

  JSON-based data storage for user accounts and progress  

  Pixel-art inspired user interface 
      

Libraries, Tools, and Frameworks Used 

        Game Engine: Godot 4.5  

        Programming Language: GDScript  

        Data Storage: JSON (for user data and progress tracking)  

        Version Control: Git & GitHub  

        Development Tools: Visual Studio Code 

 
 Steps to Install and Run the Project 

    1. Clone the Repository 

    git clone https://github.com/your-username/lexiquest.git 

    cd lexiquest 

    2. Install Godot 

    Download and install Godot Engine version 4.5 from the official website. 

     Note: Using a different version may cause compatibility issues. 

    3. Open the Project 

    Launch Godot  

    Select Import Project  

    Navigate to the cloned repository folder  

    Open the project.godot file 

    4. Run the Application 

    Open the main scene (e.g., splash.tscn or main.tscn)  

    Click the Run Project button
